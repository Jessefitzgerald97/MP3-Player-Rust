# MP3 Player (Rust)

## Overview

This is a functional graphic MP3 player that allows you to select a song from a list of MP3 files and play it. Additionally you can customize your albums and playlists.

## Features
  - Functional graphic and buttons
  - Playable audio files
  - Customizable playlists

## How to run

### Prerequisites
  - Install Rust (https://www.rust-lang.org/tools/install)

### Run Application
```bash
cargo run
